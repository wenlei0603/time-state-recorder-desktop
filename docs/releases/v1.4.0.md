# Time State Recorder Desktop v1.4.0

First public Windows desktop preview of Time State Recorder.

## Highlights

- Tauri v2 Windows desktop shell with a bundled Rust collector sidecar.
- First-run setup for local storage, capture settings, privacy defaults, and AI provider metadata.
- OpenAI-compatible provider configuration with MiniMax as a preset.
- Windows DPAPI-backed API-key storage outside normal config files.
- Tray controls for opening the app, Settings, Daily Brief, pause/resume capture, and quit.
- Launch-on-startup, start-minimized, and visible port-conflict recovery.

## Privacy

- Local-only mode is the default.
- External AI remains opt-in and shows a visible warning before provider-backed analysis is enabled.
- API keys are stored through the desktop secret store, not in `config.json`.

## Verification

The release package should be built with:

```powershell
npm run desktop:release
```

The release output includes `time-state-recorder-desktop-v1.4.0-windows-x64-setup.exe`, `RELEASE_NOTES.md`, a `.sha256` checksum file, and `release-manifest.json`.
